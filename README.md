# Three-Paper Public Reproducibility Repository

Version: `0.1.0-predoi`
Prepared: 2026-07-07

This repository staging directory consolidates the public reproducibility packages for three doctoral-publication manuscripts on electricity price forecasting, transferable load forecasting, and decision-focused virtual power plant evaluation.

## Repository Boundary

- This is a local pre-DOI staging candidate. It is not a public Zenodo/GitHub release and it does not claim a repository DOI.
- Only public or redistribution-cleared reproducibility assets from the three supplement packages are included.
- Local raw business data, school evidence, journal portal screenshots, author sign-off records, and dissertation private evidence are excluded.
- Each paper subdirectory preserves its own README, run commands, manifest, public-data notes, and license-routing files.

## Paper Packages

- `paper1_eaai/`: Engineering Applications of Artificial Intelligence route, price/load/renewable forecasting with computer-science framing.
- `paper2_ieee_access/`: IEEE Access route, transferable load forecasting and reproducible evaluation.
- `paper3_asoc/`: Applied Soft Computing route, decision-focused virtual power plant policy evaluation.

## Reproduction Entry Points

Start from each paper subdirectory, then inspect its `README.md`, `RUN_COMMANDS.sh`, `REPRODUCE_RESULTS.sh`, `MANIFEST.csv`, and `DATA_LICENSE_AND_REDISTRIBUTION.md` files. Full publication use still requires the final repository DOI release and same-day license recheck.
