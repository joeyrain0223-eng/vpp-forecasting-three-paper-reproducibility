# Data License and Redistribution Boundary

This supplement is a public-data-first reproducibility package. It is intended to support review of the manuscript's public virtual-power-plant decision-simulation claims without redistributing any local/private operational records. It also includes derived GIS context summaries and a manuscript figure, but not the raw OSM/GEM/GIS source extracts.

## Public Source

- Open Power System Data time-series package, version 2020-10-06: https://data.open-power-system-data.org/time_series/2020-10-06/time_series_60min_singleindex.csv

The bundled file `data/processed/opsd_hourly_price_load_renewables_tidy.csv` is a processed derivative used only for the reported public decision experiments. Before public repository or DOI deposit, recheck the current OPSD package metadata and preserve the source citation/attribution required by the upstream data package.

## Redistribution Boundary

- Included: processed public OPSD benchmark data, public-result CSV files, scripts, figures, derived GIS context summary tables, references, manuscript support files, and checksums.
- Excluded: raw local Hunan/Shandong operational records, customer identifiers, account numbers, contracts, settlement sheets, proprietary market-operation records, non-public trading details, raw OSM extracts, raw GEM tracker tables, and raw GIS source files.
- Do not add private files or raw GIS source extracts to the package, Git repository, journal supplement, or DOI archive unless written redistribution authorization, source attribution, and license handling are confirmed and the manuscript data-availability statement is updated accordingly.

## Final Release Gate

This file is a pre-submission control artifact. It does not replace final journal data-policy review, school recognition, author sign-off, or source-license rechecking immediately before public repository release.
