# Public Data Source Notes

The public experiments use the Open Power System Data time-series package version 2020-10-06. The working tidy file contains hourly day-ahead prices, load, wind, and solar variables for the public zones used by Paper 3.

The local/private Hunan and Shandong files are intentionally excluded from this package. The manuscript may discuss them as application evidence, but the supplement's core claims are public-data reproducible.
