# Public Data Source Notes

Paper 2 uses three public data layers:

1. UCI Electricity Load Diagrams 2011-2014 for multi-client transfer, cold-start, representation-learning, random-convolution encoder checks, trainable dilated-convolution ridge checks, multi-seed TDConv stability checks, CPU-only patch-attention transfer checks, source-trained MLP transfer checks, and neural TDConv residual-head checks.
2. Open Power System Data time-series package version 2020-10-06 for public regional load baseline evidence aligned with the other dissertation papers.
3. UCI Appliances Energy Prediction for external one-hour-ahead and multi-horizon public load forecasting sanity checks with residential appliance energy and weather covariates.

Public URLs:

- https://archive.ics.uci.edu/dataset/321/electricityloaddiagrams20112014
- https://data.open-power-system-data.org/time_series/2020-10-06/
- https://archive.ics.uci.edu/dataset/374/appliances+energy+prediction

The package bundles processed public data and result files used by the current manuscript. Local Hunan and Shandong operational records are intentionally excluded and should remain private unless explicit redistribution permission is obtained.
