# Data License and Redistribution Boundary

This supplement is a public-data-only reproducibility package. It is intended to support review of the manuscript's public transfer-learning and load-forecasting claims without redistributing any local/private operational records.

## Public Sources

- UCI Electricity Load Diagrams 2011-2014: https://archive.ics.uci.edu/dataset/321/electricityloaddiagrams20112014
- Open Power System Data time-series package, version 2020-10-06: https://data.open-power-system-data.org/time_series/2020-10-06/
- UCI Appliances Energy Prediction: https://archive.ics.uci.edu/dataset/374/appliances+energy+prediction

The bundled processed files are derived from the public sources listed above. The current UCI pages identify the two UCI datasets as CC BY 4.0 datasets. Before public repository or DOI deposit, recheck the current UCI and OPSD package pages, keep source citations/attribution with the archive, and update the repository license/readme if any upstream term changes.

## Redistribution Boundary

- Included: processed public UCI/OPSD data, public-result CSV files, scripts, figures, references, manuscript support files, and checksums.
- Excluded: raw local Hunan/Shandong operational records, customer identifiers, account numbers, contracts, settlement sheets, proprietary market-operation records, and non-public trading details.
- Do not add private files to the package, Git repository, journal supplement, or DOI archive unless written redistribution authorization exists and the manuscript data-availability statement is updated accordingly.

## Final Release Gate

This file is a pre-submission control artifact. It does not replace final journal data-policy review, school recognition, author sign-off, or source-license rechecking immediately before public repository release.
