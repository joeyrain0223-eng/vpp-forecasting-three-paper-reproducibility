# Private Data Exclusion Rule

The public repository staging package excludes all local raw data and external evidence records. In particular, do not add files from local Desktop folders named `数据`, `日信息披露`, `邵新宇`, messaging temporary folders, or any customer, settlement, contract, bid, transaction, node-price, school-recognition, portal-proof, APC, author-signoff, or dissertation-private evidence folder.

Only public reproducibility packages and their derived public-source tables may enter this repository. If a reviewer requests private evidence, provide a bounded statement, a permission-controlled appendix, or an aggregate non-identifying result after written approval rather than uploading raw files.
